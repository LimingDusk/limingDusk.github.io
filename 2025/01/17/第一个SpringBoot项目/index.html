本文将手把手带你完成第一个Spring Boot项目的创建、开发、测试与部署，并附送开发效率提升技巧。建议使用IntelliJ IDEA 2022.3以上版本跟随操作。

一、项目创建

  1. 打开IDEA，选择File > New > Project
  2. 左侧选择Spring Initializr，设置JDK版本为1.8+
  3. 项目配置示例：
    • Group: com.example
    • Artifact: demo
    • Type: Maven
    • Java Version: 8
  4. 依赖选择：
    • Spring Web（必选）
    • Thymeleaf（可选）
    • DevTools（后续讲解）

常见问题排查

Q：启动时报端口冲突？
A：在application.properties修改server.port=新端口

Q：修改代码后未生效？
A：检查devtools是否生效，可尝试Ctrl+F9手动触发编译

Q：打包时提示找不到主类？
A：确认pom.xml中正确配置了spring-boot-maven-plugin
